unicode.js
----------

This is a simple library to properly handle Unicode in various String methods.
This is basically a port of the following methods of Python's str:

* title()
* capitalize()
* casefold()
* islower()
* isupper()
* istitle()
* isspace()
* isalpha()
* isalnum()
* isdecimal()
* isdigit()
* isnumeric()
* isidentifier()
* isprintable()
* lower()
* swapcase()
* upper()
